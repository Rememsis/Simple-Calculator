# Simple-Calculator
A calculator that can do almost any calculation one by one 
Supported operations: sqrt, cbrt, round, measure, +, -, *, /, ^ (power), cos, sin, tan, acos, asin, atan, cosh, sinh, tanh, acosh, asinh, atanh, log, log10, logn, logarithm base n, %, pi, tau, e, degrees, radians, gamma, gcf, hcf, gcd, lcm, lcf, lcd.
